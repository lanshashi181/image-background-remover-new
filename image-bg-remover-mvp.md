## 1️⃣ 产品概述

### 1.1 产品定位
一款基于 Next.js 和 Remove.bg API 的轻量级图片背景移除工具，部署在 Cloudflare 边缘网络，无需存储、即传即用。

### 1.2 核心价值
- ✅ **零存储**: 图片仅在内存中处理，不落地存储
- ✅ **快速部署**: 基于 Cloudflare Pages，全球 CDN 加速
- ✅ **低成本**: 利用免费额度，初期几乎零成本
- ✅ **简单易用**: 拖拽上传，一键下载

### 1.3 目标用户
- 需要快速移除图片背景的普通用户
- 电商从业者（产品图处理）
- 设计师（快速抠图）
- 自媒体运营（素材处理）

## 3️⃣ 技术架构

### 3.1 架构图

```
┌─────────────┐      ┌─────────────────────┐      ┌──────────────┐
│   用户浏览器  │ ───→ │ Cloudflare Pages    │ ───→ │ Remove.bg API │
│  (Next.js)  │ ←─── │  (Edge Functions)   │ ←─── │  (背景移除)   │
└─────────────┘      └─────────────────────┘      └──────────────┘
                              │
                              ↓
                         （内存处理，不存储）
```

### 3.2 技术栈

| 层级 | 技术 | 版本 | 说明 |
|------|------|------|------|
| **框架** | Next.js | 15.5.2 | App Router |
| **样式** | Tailwind CSS | 4.x | 原子化 CSS |
| **语言** | TypeScript | 5.x | 类型安全 |
| **运行时** | Edge Runtime | - | Cloudflare 兼容 |
| **API** | Remove.bg | - | 背景移除服务 |
| **部署** | Cloudflare Pages | - | 全球 CDN |
| **存储** | 无 | - | 纯内存处理 |

### 3.3 数据流

```
1. 用户上传图片 → 2. Next.js API Route 接收 → 3. 转发至 Remove.bg API
                              ↓
5. 用户下载 ← 4. 返回 PNG Blob
```

### 3.4 项目结构

```
image-background-remover/
├── src/
│   ├── app/
│   │   ├── page.tsx          # 主页面（上传、预览、下载）
│   │   ├── layout.tsx        # 布局 + SEO meta
│   │   ├── globals.css       # Tailwind CSS
│   │   └── api/
│   │       └── remove-bg/
│   │           └── route.ts  # API 路由（Edge Runtime）
├── public/                   # 静态资源
├── .env.local               # 环境变量（API Key）
├── wrangler.toml            # Cloudflare 配置
├── next.config.ts           # Next.js 配置
└── package.json             # 依赖配置
```

## 4️⃣ 用户流程

### 4.1 主流程

```
访问网站 → 上传图片 → 自动处理 → 预览结果 → 下载图片
   │          │           │           │           │
   ↓          ↓           ↓           ↓           ↓
 加载页面   格式校验     调用 API    展示 PNG    浏览器下载
```

### 4.2 异常流程

| 场景 | 处理方式 |
|------|----------|
| 图片格式不支持 | 前端提示"仅支持 JPG/PNG/WebP" |
| 图片超过 25MB | 前端提示"图片大小不能超过 25MB" |
| Remove.bg API 失败 | 显示错误信息（配额用尽/网络问题） |
| 网络超时 | 提示"处理超时，请重试" |

## 7️⃣ 风险与限制

### 7.1 技术风险

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| Remove.bg API 不稳定 | 服务不可用 | 添加重试机制、错误提示 |
| Cloudflare 冷启动 | 首次请求慢 | 使用预热请求 |
| 大图片处理超时 | 请求失败 | 前端限制大小、后端设置超时 |

### 7.2 业务风险

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| API 配额用尽 | 服务中断 | 监控用量、及时升级 |
| 滥用攻击 | 成本激增 | 添加 IP 限流、验证码 |
| 版权争议 | 法律风险 | 添加使用条款、免责声明 |

### 7.3 限制说明

- 单张图片最大 25MB（Remove.bg 限制）
- 免费账户每月仅 50 张免费处理
- 不支持批量处理（MVP 阶段）
